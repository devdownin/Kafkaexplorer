Topic Exploration

Instantly browse all topics in your cluster. Get high-level metadata like partition counts and estimated data sizes. The Sampling feature pulls the latest messages so you can understand the data structure immediately.

  • Real-time metadata (offsets, partitions)

  • Quick copy to clipboard

Flink SQL Assistant

Our integrated editor powered by CodeMirror makes writing complex streaming queries easy. Use the assistant to generate JSON_VALUE or custom XmlExtract calls without looking up the syntax.

  • SQL Auto-completion (Ctrl+Space)
  • Offset Reset Switch (Earliest/Latest)
  • Visual result grid with sorting
// Generated Flink SQL 
SELECT JSON_VALUE(raw, '$.customer.name'),
       amount
FROM "orders"
WHERE amount > 100;

Stream Flow & Traceability

Trace a specific message ID (like a Correlation ID) across your entire architecture. Visualize how a message moves from the "Received" topic to "Delivered" through various transformation steps.

  • Cross-topic search by ID or Pattern
  • Support for JSONPath & XPath
  • Chronological flow visualization

Get Started in Seconds

1 Spin up Kafka (now with KRaft 4.2 support!)

Use our pre-configured Docker Compose to get a production-ready environment.

docker compose -f docker-compose-kafka4.yml up -d
2 Run the Explorer

Launch the Spring Boot application locally.

./mvnw spring-boot:run
3 Explore

Open your browser at http://localhost:8080 and start querying!
